Prework Study Guide

HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user
  • The child elements are elements within an element
  • The img child element will show you information in regards to an image on your webpage
    • All elements start and close with brackets, however the closing bracket has a slash in it

CSS

  • This CSS element allows you add style and color to your webpage
  • The CSS element codes are going to be visible to the user
  • CSS stands for Cascading Style Sheets

Git

  • Github comes from the word Git, however the letters doesn't stand for anything
    • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • Git commands allow users to create a copy of a main repository, so they can work on a webpage
  • Git is a version control system, made to track changes in files
  • Git is used to collaborate with other developers on a project

JavaScript

  • JavaScript's role is to create real-time interaction between a user and webpage
  • JavaScript is an important element in creating a webpage because like a house, the structure and color is important but is it useful? a house without a door is not a useful house. A webpage without JavaScript is usually not a useful webpage
